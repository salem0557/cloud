# بوت تيليجرام لمسح الأسهم الأمريكية والعملات الرقمية 📊

بوت يمسح **كل الأسهم الأمريكية** (NYSE + Nasdaq + AMEX، حوالي 5000 سهم) **وأهم 100 عملة رقمية** 🪙 على **فريم الساعة** بشكل **متواصل على مدار اليوم** — فور انتهاء دورة المسح تبدأ التالية مباشرة — ويرسل لك ما يحقق **3 فلاتر من 4** على الأقل:

1. 📉 السعر عند الحد السفلي لبولينجر باند (20، 2)
2. 🔻 RSI أقل من 30 (منطقة تشبع بيعي)
3. 🛡️ السعر وصل لمنطقة دعم (قيعان متكررة)
4. 📐 نموذج وتد هابط (Falling Wedge)

البوت **يدفع كل سهم فور اكتشافه** أثناء المسح — لا ينتظر اكتمال الدورة ليجمع النتائج. وهو **لا يكرر النتائج**: يرسل فقط الأسهم الجديدة أو التي تغيّرت فلاترها منذ آخر مسح.

## آلية المسح الذكية

- **قائمة مؤهلة مرتين يومياً** بترتيب اليوم التداولي: الأولى **قبل بدء الجلسة الليلية (19:30 ET — قبل 3:00 فجراً بتوقيت السعودية بنصف ساعة)** وهي أول جلسة بعد الويكند أيضاً، والثانية **بعد إغلاق الجلسة العادية (16:30 ET)**؛ بقية الوقت تُفحص القائمة المؤهلة فقط — دورات أسرع وطلبات أقل على Yahoo (الأوقات قابلة للتغيير عبر `QUALIFY_REBUILD_TIMES`)
- **قائمة ساخنة 🔥**: أي رمز يحقق فلترين من أربعة يُراقب على مسار سريع كل دقيقتين، فالإشارة التي تكتمل بين الدورات تصلك خلال دقائق (تنبيهها يبدأ برمز 🔥)
- **تهدئة تلقائية مؤقتة**: إذا بدأ Yahoo برفض الطلبات يضيف البوت مهلاً متزايدة بين الدفعات، وبمجرد التعافي تزول المهل ويعود لسرعته الكاملة تلقائياً

## مصدر البيانات

- قائمة الأسهم: ملفات Nasdaq Trader الرسمية (تُحدَّث يومياً، تُستبعد منها صناديق ETF والإصدارات التجريبية)
- الأسعار: Yahoo Finance عبر مكتبة `yfinance` — مجاني وبدون مفاتيح API
- فلتر سيولة تلقائي يستبعد الأسهم تحت 2$ أو قليلة التداول

## التشغيل

### 1. إنشاء البوت
راسل [@BotFather](https://t.me/BotFather) في تيليجرام، أرسل `/newbot`، واحفظ التوكن.

### 2. الإعداد
```bash
git clone <repo-url> && cd cloud
pip install -r requirements.txt
cp .env.example .env    # ثم ضع التوكن داخل .env
```

### 3. التشغيل
```bash
python bot.py
```

ثم افتح البوت في تيليجرام وأرسل `/start`.

> ⚠️ البوت يحتاج سيرفراً يعمل 24/7 (VPS، Railway، Render...). المسح الكامل لـ 5000 سهم يستغرق عادة 15-40 دقيقة حسب سرعة الشبكة وحدود Yahoo — وهو ضمن دورة الساعة. لن يتداخل مسحان أبداً (قفل داخلي).

### النشر على Railway

1. اربط المستودع، وتأكد أن الفرع المنشور هو الفرع الذي يحوي الكود
2. في تبويب **Variables** أضف `TELEGRAM_BOT_TOKEN`
3. **فاليوم دائم (مهم)**: من قائمة الخدمة اختر **Attach Volume** وحدد مسار التركيب مثلاً `/data` — البوت يكتشف الفاليوم تلقائياً (عبر `RAILWAY_VOLUME_MOUNT_PATH`) ويحفظ فيه المشتركين وسجل منع التكرار، فلا تفقد شيئاً عند إعادة النشر ولا تحتاج إرسال `/start` من جديد

على أي استضافة أخرى: عيّن `DATA_DIR=/path/to/persistent/dir` لنفس الغرض.

## أوامر البوت

| الأمر | الوظيفة |
|---|---|
| `/start` | تفعيل الاشتراك في التنبيهات التلقائية |
| `/scan` | مسح يدوي فوري يعرض **كل** النتائج الحالية |
| `/status` | حالة السوق والمسح وعدد المشتركين |
| `/stop` | إيقاف التنبيهات |

## شكل التنبيه

يصل التنبيه لحظة الاكتشاف أثناء المسح، والأسهم والعملات الرقمية في **رسائل منفصلة** برأس ورموز مميزة:

```
⚡ إشارة فورية — 📈 أسهم أمريكية — 14:30 ET

AAPL — 3/4 — 189.20$
  ✅ بولينجر السفلي: الحد السفلي 189.40$
  ✅ RSI تشبع بيعي: RSI=27.3
  ✅ منطقة دعم: دعم عند 187.50$
  ❌ وتد هابط: لا يوجد وتد هابط
```

```
⚡ إشارة فورية — 🪙 عملات رقمية — 14:32 ET

🪙 BTC — 3/4 — 61,250.00$
  ✅ بولينجر السفلي: الحد السفلي 61,300.00$
  ✅ RSI تشبع بيعي: RSI=28.1
  ❌ منطقة دعم: بعيد عن الدعم
  ✅ وتد هابط: وتد هابط مكتمل التكوين
```

العملات المغطاة: أهم 100 عملة تقريباً (BTC، ETH، SOL، XRP...) — أضف عملات إضافية عبر متغير `CRYPTO_EXTRA` (مثال: `CRYPTO_EXTRA=KAS,ONDO`)، أو عطّل الكريبتو كلياً بـ `CRYPTO_ENABLED=0`.

## التخصيص

كل الإعدادات قابلة للتغيير عبر متغيرات البيئة (انظر `.env.example` و `scanner/config.py`)، أهمها:

| المتغير | الافتراضي | الوصف |
|---|---|---|
| `FILTERS_REQUIRED` | 3 | عدد الفلاتر المطلوبة من أصل 4 |
| `SCAN_INTERVAL` | 1h | فريم الشموع (`1h`، `1d`...) |
| `RSI_OVERSOLD` | 30 | حد التشبع البيعي |
| `MIN_PRICE` | 2 | استبعاد الأسهم الأرخص من هذا السعر |
| `MIN_AVG_VOLUME` | 30000 | الحد الأدنى لمتوسط حجم التداول للشمعة |

## بنية المشروع

```
bot.py                  نقطة التشغيل: أوامر تيليجرام + الجدولة كل ساعة
scanner/config.py       كل الإعدادات (قابلة للتجاوز بمتغيرات البيئة)
scanner/universe.py     جلب قائمة كل الأسهم الأمريكية (مع كاش يومي)
scanner/data.py         تحميل الشموع من Yahoo على دفعات
scanner/indicators.py   الفلاتر الأربعة: بولينجر، RSI، الدعم، الوتد الهابط
scanner/engine.py       تنفيذ المسح وتجميع النتائج
scanner/state.py        حفظ المشتركين ومنع تكرار التنبيهات
```

## إخلاء مسؤولية

هذه الأداة للفحص الفني فقط وليست توصية شراء أو بيع. اكتشاف الأنماط (الدعم والوتد الهابط) يعتمد على خوارزميات تقريبية وقد يختلف عن التحليل اليدوي.
